mongoPi
====

언제까지 남이 떠준 밥만 먹고살것인가?  
이제부터는 우리스스로 밥을 떠먹자    
여기서 몽고디비를 컴파일하라.

라즈베리파이에서 몽고 디비를 컴파일해서 써본 것 정리한다.

clone 후 npm run build 로 간편하게 스크립트 실행시켜 컴파일환경 세팅부터 빌드까지 한방에 처리한다.

```
git clone https://github.com/gbox3d/mongoPi.git
cd mongoPi
npm run build
```
